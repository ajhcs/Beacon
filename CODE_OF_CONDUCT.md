# Code of Conduct

## Our pledge

We are committed to a respectful, harassment-free experience for everyone participating in this project.

## Expected behavior

- Be respectful and constructive.
- Assume good intent and ask clarifying questions.
- Accept feedback gracefully.
- Focus on technical issues, not personal attacks.

## Unacceptable behavior

- Harassment, discrimination, or intimidation.
- Personal insults or derogatory language.
- Publishing private information without permission.
- Disruptive behavior that prevents productive collaboration.

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, discussions, and review threads.

## Enforcement

Maintainers may remove comments, reject contributions, or temporarily/permanently restrict participation for violations.

## Reporting

Report incidents to project maintainers through a private channel on the repository host when available.

This policy is adapted from the Contributor Covenant spirit and intentionally kept short for this repository.
